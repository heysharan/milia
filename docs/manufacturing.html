Odoo Manufacturing

User Groups

Manufacturing user groups hierarchy from bottom to top is as follows:

  • Manufacturing: User
  • Manufacturing: Administrator

Other user groups involved are:

  • Inventory: User
  • Inventory: Administrator
  • Purchase: User
  • Quality: User
  • Employees: Officer: Manage all employees
  • Administration: Settings
  • User types: Internal User
  • Maintenance: Equipment Manager

Bill of materials can be created by manufacturing administrator.

Manufacturing users can do following operations with manufacturing order.

Manufacturing users can create raw material records.

Manufacturing users has access to work orders.

Manufacturing administrator can configure workcenters.

Equipment manager can create maintenance teams.

Equipment manager manages equipment categories, and equipments.

ERP users can raise, or can work on maintenance requests.

ERP administrator sets up manufacturing app.

Applications

Products

User can set MTO and manufacturing route on component product.

User can set buy route along with MTO on component to purchase.

User can set mapping when consumption is done in different UoM category.

Work Center

Manufacturing administrator can configure work centers under Manufacturing >> Configuration >> Work Centers. They can set operating cost per hour of the work center.

Bill of Material

Manufacturing administrator can manage BoMs under Manufacturing >> Products >> Bills of Materials. The operations can be configured on Operations page in BoM.


Manufacturing administrator can set quantity to consume in different UoM category, and that calculates quantity in main UoM category.

Operations

Manufacturing administrator can update drawings on operations under Manufacturing >> Configuration >> Operations. Previous versions can be managed through logging notes in chatter area.


Manufacturing Order

User can access manufacturing orders under Manufacturing >> Operations >> Manufacturing Orders, and can set bill of material.

User can access child MO using Child MO smart button.

User gets option to start partial production.


Work Order

Manufacturing user can access work orders under Manufacturing >> Operations >> Work Orders.

Manufacturing administrator can see time spent by operators on workorder.

Manufacturing user should wait till the work order comes into Ready stage.

Subcontracting

Manufacturing user can set Replenish on Order (MTO) and Buy route on component that needs to be subcontracted.

Manufacturing user selects Resupply Subcontractor on Order route on component to send to subcontractor.

Purchase user can set vendor (subcontractor) on product.

Manufacturing administrator can configure BoM of type Subcontracting.

On confirming MO, purchase user receives the purchase order.

Purchase user being an inventory user also can access resupply and receipt orders.

Purchase user being an inventory administrator can generate e-Waybill for components sent to subcontractor.

Purchase user being an inventory and manufacturing user can record components.


Production user can access subcontract MO under Manufacturing >> Operations >> Manufacturing Orders.

Job card for subcontractor can be managed on this manufacturing order record.

Inventory user can receive finished product by clicking Validate button on receipt order.

The receipt order can be considered as Labour Charge Acceptance Note.

Lots/Serials

Manufacturing user can generate serial numbers from order under Manufacturing >> Operations >> Manufacturing Orders.

Manufacturing user can manually set first lot number for product being produced.


Next time system follows incremental sequence for the lot of the same product produced.

Shop Floor UI

Maintenance

Equipment manager can create teams under Maintenance >> Configuration >> Maintenance Teams.

Equipment manager can create new equipment categories under Maintenance >> Configuration >> Equipment Categories.

Equipment manager can create new equipments under Maintenance >> Equipment >> Machines & Tools.

ERP user can raise any maintenance request under Maintenance >> Maintenance >> Maintenance Requests.

Configurations

ERP administrator can enable work orders, shop floor UI, and subcontracting feature under Manufacturing >> Configuration >> Settings.



ERP administrator can enable custom worksheets for maintenance requests under Maintenance >> Configuration >> Settings.
