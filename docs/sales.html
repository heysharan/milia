User Groups

Sales team hierarchy from bottom to top is as follows:

  • Sales: User: All Documents
  • Sales: Administrator

Other user groups involved are:

  • Inventory: User
  • Inventory: Administrator
  • Administration: Settings

Sales user should be still present in Inventory / User group to perform certain operation on product master.

Application-wise steps

CRM

Access CRM app from main dashboard

Create new lead by clicking New button in CRM kanban view, and add label, phone, and email

Set all information under Extra Information page as it is further used when creating new customer

Create Quotation from Lead, which also gives option to create a new customer record


Mark lead as lost


Document additional notes on lead record

Sales

Access Sales app from main dashboard

Direct sales order can be created using New button

Add order lines by clicking Add a product link

Set quantity in order line

Create new products to later configure in order lines

Set lead times on sales order lines

Set shipping policy under Other Info page based on which delivery gets scheduled

Sales user can send PRO-FORMA invoice to customer.

Sales user can access delivery records from below highlighted smart button

Send request to production, and check production activities from sales order


Cancel a quotation

Delivery Orders

Delivery can be split for products by updating quantity on moves, and creating backorder at time of validating delivery



The backorder gets updated delivery date

Do return for exchange by clicking Return button

Note: Sales user should check T&Cs on sales quotation for warranty related information

Scrap return



Product

To sales user, product master is accessible under:

  1. Sales >> Products >> Products
  2. Inventory >> Products >> Products

Set product name

Set sales price after getting details from manufacturing department, this helps in getting price for repeat orders with same drawing specification

Set HSN code

Set sales tax

Set track inventory by unique serial number/lots

Set unique internal reference based on unique drawing number which further helps in managing repeat orders

Select Manufacture route along with MTO under Inventory page

Set Customer Lead Time

Customer

Customer records are accessible under Contacts app

To sales user, they are also accessible under:

  1. CRM >> Sales >> Customers
  2. Sales >> Orders >> Customers

Create company type customer record with required tax details

Set name, email, phone, and other details

Create multiple point of contacts, invoice and delivery addresses for one customer

Select fiscal position if applicable to customer

Pricelists

Sales user can create new foreign currency pricelists under Sales >> Products >> Pricelists menu

This pricelist can be further used to draft quote in foreign currency

ERP Administrator Settings

Sales administrator configures sales team.

ERP administrator can set global security lead time under Inventory >> Configuration >> Settings menu, helpful in scheduling deliveries.

ERP administrator sets up alias domain.

ERP administrator configures incoming mail server.

ERP administrator enables Pro-Forma invoices feature.

The ERP administrator sets the sales order sequence implementation to No gap.
